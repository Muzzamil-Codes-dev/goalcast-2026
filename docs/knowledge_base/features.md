# Feature Glossary

- **Elo rating**: a self-updating strength score. Teams gain points for beating stronger
  opponents and lose points for losing to weaker ones; margin of victory and match
  importance scale the update. Computed chronologically so it never sees the future.
- **Form**: average points per game over the last 5 matches (3 for a win, 1 for a draw).
- **Rolling goals for / against**: recent scoring and conceding rates.
- **Rest days**: days since each team's previous match.
- **Home advantage**: a boost for the home team; 2026 co-hosts (USA, Canada, Mexico) get an
  additional edge even at neutral venues.
